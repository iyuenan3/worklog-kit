# CORE

## 身份
本仓是我的「父项目」vault：孵化器 + Wiki + 日记，所有子项目的每日动态与长期知识在此汇流（方法论见 `../docs/methodology.md`）。

## Non-Goals
- 不是代码仓：不放任何子项目源码
- 不是公开博客：默认私有，不面向读者写作

## 绝不（硬约束）
- `diaries/` 只追加，绝不覆盖 / 重写历史日记
- 凭证 / 私钥 / token 绝不入 git（.gitignore 兜底 + lint 扫描）
- IM 数据源默认只记本人发出的消息
- wiki 契约锚点（index 三段标题 / log 锚点注释 / todos 分区标题）不删

## 生命周期
active（个人长期运行，无终点）
