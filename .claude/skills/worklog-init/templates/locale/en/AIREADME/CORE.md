# CORE

## Identity
This repo is my "parent project" vault: incubator + wiki + diary, where the daily activity and long-term knowledge of all my projects converge (methodology: `../docs/methodology.md`).

## Non-Goals
- Not a code repo: no subproject source code lives here
- Not a public blog: private by default, not written for an audience

## Never (hard constraints)
- `diaries/` is append-only; never overwrite or rewrite past entries
- Credentials / private keys / tokens never enter git (.gitignore backstop + lint scan)
- IM sources record only my own outgoing messages by default
- Wiki contract anchors (index's three headings / log's anchor comment / todos' section headings) must not be deleted

## Lifecycle
active (personal, long-running, no end state)
